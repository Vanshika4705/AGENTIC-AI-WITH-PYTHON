# Day 14 – MongoDB CRUD Operations and Introduction to Streamlit

**Date:** 13th July 2026

---

## Overview

The fourteenth day of the Agentic AI Training focused on implementing **CRUD (Create, Read, Update, Delete)** operations in MongoDB using Python and exploring **Streamlit**, a powerful Python framework for building interactive web applications. The session provided practical knowledge of updating and deleting documents stored in a MongoDB database while reinforcing previously learned database concepts. Along with backend operations, we were introduced to Streamlit, which enables developers to create attractive user interfaces directly with Python. To apply these concepts in a real-world scenario, we designed and implemented a simple **User Registration Form** using Streamlit components.

---

## Learning Highlights

### Topics Covered

* MongoDB CRUD Operations
* Updating Documents in MongoDB
* Deleting Documents from MongoDB
* Introduction to Streamlit
* Streamlit Components
* Building Interactive User Interfaces with Python
* User Registration Form using Streamlit

---

### 1. MongoDB CRUD Operations

* Revised the concept of **CRUD (Create, Read, Update, Delete)** operations and their importance in database management.
* Learned how to modify and manage existing documents stored in MongoDB collections.
* Understood how CRUD operations form the foundation of almost every database-driven application.
* Discussed practical examples where updating and deleting records are necessary to keep information accurate and up to date.
* Explored how MongoDB's flexible document structure simplifies data management compared to traditional relational databases.

---

### 2. Updating Documents in MongoDB

* Learned how to update MongoDB documents using Python and the PyMongo library.
* Understood how filters help identify the exact document that needs modification.
* Practiced updating both individual documents and multiple documents within a collection.
* Verified the updated records by retrieving data after performing update operations.
* Learned the significance of maintaining data consistency while modifying database records.

---

### 3. Deleting Documents from MongoDB

* Learned how to remove unwanted documents from a MongoDB collection using Python.
* Understood the difference between deleting a single document and deleting multiple matching documents.
* Discussed why filters should always be applied carefully before executing delete operations.
* Verified that records were successfully removed from the database after deletion.
* Understood how delete operations help maintain clean, relevant, and well-organized datasets.

---

### 4. Introduction to Streamlit

* Learned about **Streamlit**, an open-source Python framework designed for creating interactive web applications.
* Understood that Streamlit enables developers to build user interfaces entirely with Python without requiring HTML, CSS, or JavaScript.
* Explored its key features, including rapid development, minimal code, automatic updates, and easy deployment.
* Discussed its applications in Artificial Intelligence, Machine Learning, Data Science, analytics dashboards, and data visualization.
* Learned how Streamlit bridges the gap between backend logic and frontend presentation, making application development much faster.

---

### 5. Basic Streamlit Components

* Gained hands-on experience with several built-in Streamlit components.
* Learned how to create:

  * Titles and headings
  * Text input fields
  * Password input fields
  * Buttons
  * Labels
  * Basic layout elements
* Understood how these components work together to create clean and user-friendly interfaces.
* Learned how Streamlit automatically refreshes the application whenever the Python code is modified, making development efficient.

---

### 6. User Registration Form

As a practical implementation, we developed a **User Registration Form** using Streamlit.

The application included:

* User name input field.
* Email address input.
* Password field for secure input.
* Submit button for form submission.
* Displaying user-entered information after successful submission.

This exercise demonstrated how quickly an interactive web application can be created using Python. It also provided practical experience in handling user inputs and displaying dynamic output through Streamlit components.
