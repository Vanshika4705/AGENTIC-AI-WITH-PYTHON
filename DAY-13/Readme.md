# Day 13 – MongoDB Setup, Database Connection, and Data Insertion with Python

**Date:** 10th July 2026

---

## Overview

The thirteenth day of the Agentic AI Training focused on **MongoDB**, a widely used NoSQL database for developing modern applications. The session covered the core concepts of MongoDB, its document-based architecture, and how it differs from traditional relational databases. We also learned how to set up a MongoDB Atlas cluster, configure database access, connect MongoDB with Python, and insert data into the database using Python programs.

## 1. Introduction to MongoDB

* Understood the basics of **MongoDB** as a **NoSQL database**.
* Learned that MongoDB stores information as **documents** rather than tables consisting of rows and columns.
* Explored its key features such as scalability, flexibility, and efficient performance.
* Discussed real-world applications of MongoDB in web development, AI systems, and cloud platforms.
* Learned that each document can have a different structure, making MongoDB highly adaptable.
* Compared MongoDB with relational databases to understand when NoSQL databases are preferred.

## 2. MongoDB Atlas Setup

* Created a new project and deployed a cloud-based database cluster.
* Understood the purpose of database clusters and their role in cloud storage.
* Configured network permissions and database users for secure database access.
* Explored the Atlas dashboard and learned how cloud databases are managed online.
* Understood the importance of selecting the appropriate cloud provider and region during setup.

## 3. Connecting Python with MongoDB

* Connected a Python application with the MongoDB database.

* Verified that the connection was established successfully through Python code.

## 4. Creating Databases and Collections

* Learned how MongoDB organizes data using **Databases** and **Collections**.
