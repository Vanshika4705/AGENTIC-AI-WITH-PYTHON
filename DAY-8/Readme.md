# Day 8 – Object-Oriented Programming (OOP) in Python

**Date**: 4th July 2026

## Overview

The eighth day of the Agentic AI Training focused on the fundamentals of Object-Oriented Programming (OOP) in Python. The session introduced core OOP concepts such as classes, objects, constructors, and reference variables, and demonstrated how they can be used to represent real-world entities in software applications. During the class, we implemented a basic Restaurant Management System using OOP. For self-practice, I independently designed a YouTube System to strengthen my understanding of object-oriented design and implementation.

---

## Learning Highlights

### Topics Covered

- Introduction to Object-Oriented Programming (OOP)
- Classes and Objects
- Real-World vs Computer Science Representation
- Constructors (`__init__()`)
- Reference Variables
- Object Creation
- Restaurant Management System using OOP
- YouTube System Design using OOP

---

### 1. Introduction to Object-Oriented Programming

- Learned the fundamentals of Object-Oriented Programming (OOP) and its role in modern software development.
- Understood how OOP organizes programs into reusable and modular components.
- Explored the advantages of OOP, including better code organization, scalability, maintainability, and reusability.
- Discussed how OOP simplifies the development of large and complex applications.

---

### 2. Classes and Objects

- Understood that a class acts as a blueprint for creating objects.
- Learned that an object is an instance of a class with its own attributes and methods.
- Explored how classes define the properties (attributes) and behaviors (methods) of objects.
- Practiced creating objects and accessing their data through reference variables.

---

### 3. Real-World and Computer Science Applications

- Learned how real-world entities can be represented as objects in programming.
- Identified classes and objects from various real-life scenarios.
- Discussed examples such as students, vehicles, restaurants, and online platforms.
- Understood how OOP helps in designing organized and scalable software systems.

---

### 4. Constructors in Python

- Learned the purpose of constructors during object creation.
- Understood how the `__init__()` method initializes object attributes automatically.
- Explored how constructors simplify object initialization and improve code readability.
- Observed how constructors help create well-structured and maintainable programs.

---

### 5. Reference Variables

- Learned the concept of reference variables in Python.
- Understood that reference variables store references to objects rather than the actual objects.
- Explored how multiple reference variables can point to the same object.
- Learned how object attributes and methods are accessed through reference variables.
