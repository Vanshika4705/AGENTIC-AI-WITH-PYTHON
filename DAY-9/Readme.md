# Day 9 – Constructors and Circular Linked Lists in Python

**Date:** 6th July 2026

## Overview

The ninth day of the Agentic AI Training focused on expanding our knowledge of Object-Oriented Programming and data structures in Python. The session covered different approaches to creating constructors and explained how constructors can be customized to initialize objects based on different requirements. We also learned the implementation of a **Circular Linked List** through a Song Playlist example. As part of my self-practice, I developed a **WhatsApp Message Management System** using a Circular Linked List and implemented several insertion and deletion operations to strengthen my understanding.

- Song Playlist Implementation

### 1. Constructors in Python

- Revisited the concept of constructors and their importance in object initialization.
- Explored various ways of defining constructors in Python.
- Learned how default values, parameterized constructors, and flexible initialization techniques can be used while creating objects.
- Understood how constructors automatically initialize object attributes, making programs more organized and easier to manage.
- Practiced multiple examples to understand the behavior of constructors in different situations.

---

### 2. Circular Linked List

- Learned about the **Circular Linked List**, a linked list in which the last node points back to the first node instead of ending with a `NULL` reference.
- Understood how this circular connection enables continuous traversal through the list.
- Studied the structure of nodes consisting of data and a reference to the next node.
- Explored the advantages and practical use cases of Circular Linked Lists in real-world applications.

---

### 3. Song Playlist Implementation

- Developed a **Song Playlist** using a Circular Linked List.
- Learned how each song is represented as an individual node connected in a circular sequence.
- Understood how continuous music playback can be achieved by repeatedly traversing the circular list.
- Practiced insertion and deletion operations while preserving the circular structure of the linked list.

### The implementation included:
